# Employee Dashboard App

An Android application for managing employees.

## Features
- Admin Login
- Add Employee
- Update Employee
- Delete Employee
- Employee Dashboard
- Room Database
- RecyclerView UI

## Tech Stack
- Kotlin
- Room Database
- Android Studio
- Material UI
